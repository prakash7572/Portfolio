# Portfolio Website

A personal portfolio website built with **React** for the frontend and **.NET Core** for backend services. This project includes modules like **Resume**, **Portfolio**, **Pricing**, **About**, and **Contact** with email functionality for inquiries.


## Features
- **Modular Design** with a responsive layout
- **Email Functionality**: Users can send inquiries, handled by a .NET Core API
- **RESTful API**: Backend efficiently manages requests
